# Naive Bayes Classifier Shiny App

This is an interactive web application built with R and Shiny that allows users to upload custom datasets, visualize the relationship between target classes and features, train a Naive Bayes classification model, and evaluate performance.

## Features
- **Dataset Preview:** Instantly view uploaded CSV data in an interactive table.
- **Feature Exploration:** Dynamically plot target variables against any selected feature to inspect distributions.
- **Model Training:** Automated 80/20 data partitioning and Naive Bayes model training.
- **Performance Evaluation:** Inspect confusion matrices and key model metrics.
